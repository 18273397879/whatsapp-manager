# WhatsApp 分流管理系统

一个简单的 WhatsApp 号码分流系统，可以将访客随机分配到不同的 WhatsApp 号码。

## 功能特点

- 支持多个 WhatsApp 号码分组
- 随机分配访客
- IP 访问限制（24小时内同一IP限访3次）
- 链接有效期管理
- 访问统计
- 批量导入号码

## 使用方法

1. 访问管理后台：`你的域名/admin`
2. 登录账号：
   - 用户名：123
   - 密码：123
3. 创建新的分流链接
4. 分享生成的短链接给客户

## 技术栈

- Python Flask
- SQLite 数据库
- Vercel 部署

## 部署说明

本项目已通过 Vercel 自动部署。
